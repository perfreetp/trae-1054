# 智慧港口安全环保监控桌面客户端

## 项目概述

基于 Electron + React + TypeScript + Ant Design 构建的智慧港口安全环保监控系统桌面客户端，供安全管理室日常值守和应急联动使用。

## 功能模块

### 1. 港区地图
- 电子地图展示港区全貌
- 危险区域标注（红色高危区、橙色警示区）
- 人员实时位置分布
- 车辆实时位置追踪
- 区域/人员/车辆图层开关控制

### 2. 风险点清单
- 风险点台账管理（高/中/低风险）
- 隐患登记与整改跟踪
- 巡查路线设置与管理
- 支持隐患下发、验收、闭环管理

### 3. 作业许可
- 动火作业、高处作业、受限空间、临时用电等许可管理
- 作业申请、审核、批准、驳回流程
- 安全措施确认
- 作业时间和地点管理

### 4. 视频巡查
- 多路视频监控墙（支持1/4/9/16画面布局）
- 摄像头分组管理
- 实时监控状态显示
- 巡查任务管理
- 异常告警记录

### 5. 环境监测
- 粉尘浓度实时监测（含PM2.5/PM10）
- 噪声监测趋势分析
- 温湿度监测
- 污水/油污异常记录
- 数据可视化图表展示

### 6. 事件处置
- 突发事件上报（分级分类）
- 应急预案库管理
- 一键启动应急预案
- 快速通知相关岗位
- 处置过程全程记录
- 事件状态跟踪（上报→处置→解决→结案）

### 7. 复盘报表
- 安全月报自动生成
- 隐患类型统计分析
- 安全趋势图表展示
- 未闭环隐患清单
- 报告导出打印

## 技术栈

- **前端框架**: React 18 + TypeScript
- **UI组件库**: Ant Design 5
- **地图组件**: Leaflet + react-leaflet
- **图表库**: ECharts + echarts-for-react
- **桌面框架**: Electron
- **构建工具**: Vite
- **路由**: React Router DOM
- **日期处理**: Day.js

## 主要功能说明

### 安全值守日常使用流程

1. **登录系统** → 进入港区地图总览
2. **查看实时状态** → 人员/车辆分布、危险区域
3. **风险巡查** → 通过视频巡查模块查看各区域
4. **作业审核** → 处理待审核的作业许可申请
5. **环境监控** → 关注粉尘、噪声等环境数据
6. **隐患处理** → 登记发现的隐患，下发整改任务

### 应急联动处置流程

1. **事件上报** → 发现突发事件立即上报
2. **启动预案** → 根据事件类型选择对应应急预案
3. **通知岗位** → 一键通知相关责任部门和人员
4. **跟踪处置** → 记录每一步处置过程
5. **结案归档** → 事件解决后完成结案
6. **复盘分析** → 在报表模块进行统计分析

## 数据说明

当前版本使用模拟数据进行演示，实际使用时需对接以下系统接口：
- 人员定位系统
- 车辆GPS系统
- 视频监控系统（NVR/摄像头SDK）
- 环境监测传感器
- 作业许可审批系统
- 应急指挥系统
